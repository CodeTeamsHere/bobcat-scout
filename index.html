HOW TO USE BOBCAT SCOUT

        
      

      


        

          
Interactive Walkthrough

Quick Start

          
    
            
  1. Tap the maroon mic button and describe the match in plain English. You can also type into the box.
    2. 
            
  2. Tap AUTO-FILL FIELDS. The app parses your description into the form.
    3. 
            
  3. Review the fields (Step 2) — anything with a green AI badge was auto-filled. Edit anything that's wrong.
    4. 
            
  4. Fill in any required fields marked with *. The Generate button stays disabled until they're filled.
    5. 
            
  5. Tap GENERATE, then either scan the QR code, or — if a Sheet is connected — tap SUBMIT TO SHEET.
    6. 
            
  6. Tap SAVE & NEXT MATCH to keep going (it also submits to the Sheet when connected). Match number increments automatically.
    7. 
          

        


        

          
For the Host (spreadsheet owner) — one-time setup

          
The host is whoever owns the scouting Google Sheet. Do this once so scouters' matches flow straight into your Sheet.

          
    
            
  1. Create a Google Sheet, add the script, and deploy it as a Web App — full click-by-click steps are in the project's SETUP-SHEET.md (and inside apps-script/Code.gs).
    2. 
            
  2. In the Sheet's Config tab, set a Passcode (and optionally an Active Event + dates, so it only accepts data during your event).
    3. 
            
  3. Open this app → ⚙ SHEET → paste your Web App URL + passcode → SAVE → SEND TEST ROW to confirm it works.
    4. 
            
  4. Tap COPY SCOUT LINK and share that one link with your scouters — opening it connects their phones automatically.
    5. 
          

        


        

          
For Scouters (filling in matches)

          
    
            
  1. Connect once: open the link your host sent (or tap ⚙ SHEET and paste the URL + passcode they give you). The dot turns green when connected — you only do this one time.
    2. 
            
  2. Scout a match: describe it → AUTO-FILL → fix anything → GENERATE.
    3. 
            
  3. Send it: tap SUBMIT TO SHEET (or SAVE & NEXT, which also submits). Your row appears in the host's Sheet right away.
    4. 
            
  4. Keep going for every match: tap SAVE & NEXT MATCH — the form clears and the match number bumps up automatically, ready for your next one. Scout as many matches as you want; there is no limit and submitting one never locks you out.
    5. 
            
  5. No signal? The app saves it and shows SHEET (n) — it sends automatically once you're back online. The QR code always works as a backup too.
    6. 
          

          

            You scout match after match — one scouter sends many rows. Each match is saved as its own row in the host's Sheet. The duplicate-check only matters if you re-send the exact same match and team; then it updates that single row so nothing is doubled. Different matches (or different teams) are always brand-new rows.
          

        


        

          
Microphone Troubleshooting

          
Voice input uses your browser's built-in speech recognition. If the mic button doesn't work or no text appears, follow the directions for your device below.


          

            
📱 Android (Chrome)

            
    
              
  1. Tap the lock icon (🔒) to the left of the address bar.
    2. 
              
  2. Tap Permissions → Microphone → set to Allow.
    3. 
              
  3. Reload the page and tap the mic button again.
    4. 
              
  4. If still blocked: open Android Settings → Apps → Chrome → Permissions → Microphone → Allow.
    5. 
              
  5. Make sure no other app (Zoom, Meet, voice recorder) is using the mic.
    6. 
            

          


          

            
🍎 iPhone / iPad (Safari)

            
    
              
  1. Voice input requires Safari on iOS 14.5 or newer. Chrome on iOS does not support it — open this page in Safari.
    2. 
              
  2. When you tap the mic, iOS will ask for microphone permission. Tap Allow.
    3. 
              
  3. If you accidentally denied it: open Settings → Safari → Microphone → set this site to Allow.
    4. 
              
  4. Or: Settings → Privacy & Security → Microphone → make sure Safari is toggled on.
    5. 
              
  5. Reload the page after changing the setting.
    6. 
            

          


          

            
💻 Computer (Chrome / Edge / Safari on macOS)

            
    
              
  1. Click the lock icon or tune icon (🔒 / ⚙️) to the left of the address bar.
    2. 
              
  2. Find Microphone and set it to Allow.
    3. 
              
  3. On macOS, also open System Settings → Privacy & Security → Microphone and make sure your browser is enabled.
    4. 
              
  4. On Windows, open Settings → Privacy & security → Microphone and confirm "Let apps access your microphone" is on.
    5. 
              
  5. Firefox: voice recognition is not supported. Use Chrome, Edge, or Safari instead — or just type into the box.
    6. 
              
  6. Reload the page after changing any permission.
    7. 
            

          


          

            Still not working? You can always type the match description directly into the text box — every voice feature works the same way with typed text.
          

        


        

          
Tips

          
    
            
  • Required fields are marked with *. The rest are optional.
    • 
            
  • Quality words help the parser. Say "passing was good" → 5, "pickup was pretty good" → 4, "bad pickup" → 2.
    • 
            
  • If the team didn't show up, toggle No Show in the Prematch section.
    • 
            
  • Your scout name and event key are remembered between sessions.
    • 
            
  • Works offline. Open the app once with a connection, then it keeps working with no signal. On your phone, use the browser menu → Add to Home Screen to install it like an app.
    • 
            
  • Auto-saved. If the page reloads mid-match, your entries come back. Use SAVE QR AS IMAGE (QR tab) to keep a backup of any match.
    •

Auto team numbers (optional)

        
Load the event's match schedule so the team number auto-fills from match #, alliance, and station. Needs a free Blue Alliance API key and the Event Key set in the form behind this dialog.

        
        
        

          
        

        


        

        
Google sign-in (max security, optional)

        
Locks submissions to your team's Google accounts. The host pastes an OAuth Client ID (steps in SETUP-SHEET.md) and sets the Sheet's Config tab; scouters then sign in below. Leave blank to use the passcode only.

        
        
        

          
          
        

        

        


        

          Don't have the URL yet? The Sheet owner sets this up once — see the apps-script/Code.gs file in the repo (it has click-by-click steps) or the SETUP-SHEET.md guide. Then use COPY SCOUT LINK to send every scout a single link that connects their app automatically.
